**It**'s about dividing the code into tokens.
- removes comments
- saves text of identifiers, strings, and numbers
- tags tokens with line numbers (for error messages)


# The Role of a Lexical Analyzer
- It divides the raw source code into a stream of tokens by reading characters from a buffer (4KB at a time)
- Always take the longest possible token from the input (read "while", and not just "w" or "wh")
- Remove comments, and whitespaces from the source code
- Uses 2 pointers (lexBegin : start of the current leceme), and also, we might need to lookahead at atleast 1 char to decide a lexeme.
- Saves lexeme info (text of identifiers, strings, numbers)

Patterns are defined by regular expressions. We need to use DFA's.
lex(), scangen, etc, build the DFA automatically, from a set of regex

> [!info] Regex
> digit $\rightarrow$ 0 | 1 | ... | 9 = [0 - 9]
> letter $\rightarrow$ A | B | ... | Z | a | b | ... | z = [A-Za-z]
> integer $\rightarrow digit^{+}$
> exp $\rightarrow (e|E)(+ | - | \epsilon)$ integer
> decimal $\rightarrow digit^{*}$(. digit | digit .) $digit^{*}$
> real $\rightarrow$ integer exp | decimal (exp | $\epsilon$)
> num $\rightarrow$ integer | real

# Scanner Implementation
## Ad-Hoc
Like the one we did in the lab. 
```c
if (isalpha(c) || c == '_') {
    // read identifier / keyword
    while (isalnum(peek()) || peek() == '_') advance();
    return IDENTIFIER;
}
else if (isdigit(c)) {
    // read integer
    while (isdigit(peek())) advance();
    return NUMBER;
}
else if (c == '/') {
    if (peek() == '/') skipLineComment();
    else return SLASH;
}
```

Doesn't think in terms of automata at all. We just write procedural code, that checks the characters and consumes input.

## Semi Mechanical Pure DFA
Writing pure DFA as a set of nested case statements (like, by using states...(in switch/case)).

We have to manually write the DFA in this case though. It's structured and formal, but isn't generated by tools

```c
int state = 0;

while (true) {
    char c = peek();

    switch (state) {
        case 0:
            if (isalpha(c)) { state = 1; advance(); }
            else if (isdigit(c)) { state = 2; advance(); }
            else error();
            break;

        case 1:
            if (isalnum(c)) advance();
            else return IDENTIFIER;
            break;

        case 2:
            if (isdigit(c)) advance();
            else return NUMBER;
            break;
    }
}
```
